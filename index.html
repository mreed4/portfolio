Club Struttgart

The intent of this project was to create a responsive page or site for a made up or actual "club". I personally love Porsches, so I made up a local car club. Unfortunately and unsurprisingly, I do not own a Porsche … . The color scheme is inspired by the Porsche brand colors. Overall still needs some work in terms of its completedness, but I think it checked all the boxes.

Check it out Live, or browse the Repo. If you have a Codecademy account, check out the Project page or the forum thread about the project.

Design System

Here I was to create a "design system"—perhaps something like Bootstrap's design language—to show that I know CSS and basic design principles, including colors and typography. I wanted the presentation page itself to be something other than white, so I went with a gray and light gray to show off the elements. This page is not responsive—yet.

Check it out Live, or browse the Repo. If you have a Codecademy account, check out the Project page or the forum thread about the project.
